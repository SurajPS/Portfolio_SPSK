# Suraj Plachikaad Suresh Kumar — Portfolio

A clean, classy, multi-page portfolio site (Home, Experience, Education, Skills)
with a light/dark theme toggle. Built with plain HTML/CSS/JS — no build tools.

## Maintain it with ease
Edit **one** file for almost everything:
- `assets/js/content.js` — your name, intro, hobbies, jobs, education, skills.

Anything marked `TODO` is a placeholder — replace it with your real text.

### Add a photo
1. Drop the image in the `images/` folder.
2. In `content.js`, set the filename, e.g. `photo: "images/profile.jpg"`.
3. For the Home gallery, list filenames in `home.gallery`.

## Preview locally
Open `index.html` in a browser. (For loading images correctly you can also run
a tiny server: `python -m http.server` then visit http://localhost:8000.)

## Host on GitHub Pages
1. Create a repo and upload these files (or `git init` + push).
2. Repo → Settings → Pages → Source: `main` branch, folder `/root`.
3. Your site goes live at `https://<username>.github.io/<repo>/`.

## Pages
